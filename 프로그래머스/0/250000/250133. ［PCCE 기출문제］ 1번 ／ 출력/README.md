빈칸을 제외한 기본 코드는 수정할 수 없습니다.
빈칸을 채우지 않을 경우, 실행 결과에 에러 메시지가 표시됩니다.

```python
string_msg = "Spring is beginning"

int_val = 3

string_val = "3"

print(string_msg)
print(int_val + 10)
print(string_val + "10")
```
